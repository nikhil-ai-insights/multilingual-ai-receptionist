## 🎯 The Problem

Every missed call is a missed customer. Every language barrier is a lost sale. Every manual booking is a wasted hour.

| Pain Point | Impact |
|---|---|
| 📵 Calls missed during busy hours | Lost revenue & leads |
| 🌙 No staff after working hours | 0% after-hours conversion |
| 📋 Manual booking process | Delays & double bookings |
| 🗣️ Language barriers | Reduced reach & trust |
| ⏳ Slow response times | Poor customer experience |

**The Multilingual AI Receptionist eliminates every single one of these.**

---

## 💡 The Solution

An AI-powered virtual receptionist that answers calls instantly, understands your customers, books appointments in real time, and sends confirmations — all without any human intervention.

```
Customer Calls → AI Answers → Books Appointment → Sends Confirmation
                                    ↑
              Checks live calendar availability in real time
```

## ⚙️ How It Works

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  📞 STEP 1        Customer dials the business phone number      │
│                                                                 │
│  🤖 STEP 2        AI Receptionist answers within seconds,       │
│                   greets professionally in customer's language  │
│                                                                 │
│  🗣️  STEP 3        AI collects requirements:                     │
│                   → Service needed                              │
│                   → Preferred date & time                       │
│                   → Name & contact details                      │
│                                                                 │
│  📅 STEP 4        System checks live calendar availability      │
│                                                                 │
│  ✅ STEP 5        Slot available? Appointment booked!           │
│                   Confirmation sent via SMS & Email             │
│                   Customer database updated automatically       │
│                                                                 │
│  🔄 STEP 6        Customer can call anytime to reschedule,      │
│                   cancel, or check booking details              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn
- API keys: OpenAI, Twilio, SendGrid, Google Calendar
- Database: Supabase / PostgreSQL instance

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/nikhil-ai-insights/multilingual-ai-receptionist.git
cd multilingual-ai-receptionist
```

**2. Install dependencies**
```bash
npm install
```

**3. Configure environment variables**

Create a `.env` file in the root directory:
```env
# AI & Core
OPENAI_API_KEY=your_openai_key

# Telephony
TWILIO_ACCOUNT_SID=your_twilio_sid
TWILIO_AUTH_TOKEN=your_twilio_token
TWILIO_PHONE_NUMBER=your_twilio_number

# Database
DATABASE_URL=your_supabase_or_postgres_url

# Calendar
GOOGLE_CALENDAR_CLIENT_ID=your_google_client_id
GOOGLE_CALENDAR_CLIENT_SECRET=your_google_client_secret

# Notifications
SENDGRID_API_KEY=your_sendgrid_key
```

**4. Start the backend server**
```bash
npm run server
```

**5. Start the frontend dashboard**
```bash
npm run dev
```

Visit `http://localhost:3000` to access the admin dashboard.
